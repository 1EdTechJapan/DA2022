容物について

このサンプルコードはOneRoster Japan Profile v1.2の様式にしたがって作成されたCSVファイルを2023年3月時点での弊社のデータベース構造に反映させるためのサンプル実装です(実際のテーブル名およびフィールド名はセキュリティ目的で変更されています)。
OneRosterの構文解析は、org.apache.commons:commons-csvを利用してCSVファイルを構文解析しています。

注意点
このサンプルコードは弊社の実装するOneRoster連携プログラムの一部であるため、サンプルコード単体で実行することは出来ません。

環境
Java 8
MongoDB 3.6.5
